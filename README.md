Use Swagger UI in your Trails project

## Install

```sh
$ npm install --save trailpack-swagger
```

## Configure

```js
// config/main.js
module.exports = {
  packs: [
    // ... other trailpacks
    require('trailpack-swagger')
  ]
}
```
